Configurar Vue SPA #

Se supone que tiene algo de experiencia trabajando con VueJS, su enrutador y el paquete de administración de estado Pinia. Puede instalar el Vite usando:

// with npm
$ npm install -D vitest

// or with yarn
$ yarn add -D vitest

// or with pnpm
$ pnpm add -D vitest

Instale los siguientes paquetes:

  • TypeScript

  • Pinia

  • Vue Router

  • Axios

  • Tailwind CSS

  • Vitest

  • Vue Test Utils

  • Vue Testing Library

  • Jest Dom Testing Library
